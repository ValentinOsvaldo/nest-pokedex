# Execute in dev mode

1. Clonar this repository
2. Execute ```pnpm install```
3. Must be have nest CLI installed
```
npm i -g @nestjs/cli
```
4. Start database in your local
```
docker-compose up -d
```

## Stack

* Nestjs
* Mongo
* Docker
